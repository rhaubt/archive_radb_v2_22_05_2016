== Description ==

Shows the number of post or/and pages of a certain state in the menu (similar to the available plugin updates).

== Installation ==
1. Unpack the download package

2. Upload folder include all files to the `/wp-content/plugins/` directory.

3. Activate the plugin through the `Plugins` menu in WordPress

3. Go to `Options` > `Admin Menu Post/Page Counter` menu and define what to display
